# Developer Setup Checklist

## Environment
- [ ] Node.js installed (version X.X)
- [ ] Package manager configured (npm/yarn/pnpm)
- [ ] IDE/Editor configured

## Project Setup
- [ ] Repository cloned
- [ ] Dependencies installed
- [ ] Environment variables configured
- [ ] Local database/services running

## Verification
- [ ] Tests pass locally
- [ ] Application starts successfully
- [ ] Can connect to required services

## Tools
- [ ] Claude Code installed
- [ ] Required MCP servers configured
- [ ] Git hooks installed
